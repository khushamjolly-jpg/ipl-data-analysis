# 🏏 IPL Data Analysis

Exploratory data analysis on IPL cricket data using Python and Pandas.

## 📊 Analyses Performed
- **Top Run Scorers** — All-time leading run scorers across IPL seasons
- **Top Wicket Takers** — Bowlers with most wickets (excluding run-outs)
- **Team Win Percentage** — Win rates across all franchises
- **Toss Impact Analysis** — Does winning the toss help win the match?

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- JupyterLab

## 📁 Dataset
IPL dataset sourced from [Kaggle](https://www.kaggle.com/datasets/ramjidoolla/ipl-data-set)
